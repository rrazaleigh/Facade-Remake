# Facade Remake

An attempt at making a game inspired by Facade, with AI-driven dialogue and story progression.

Built with Godot 4 and Ollama (llama3.2).

## Setup

1. Install [Ollama](https://ollama.ai) and pull a model:
   ```
   ollama pull llama3.2:3b
   ```
2. Open the project in Godot 4.
3. Run the game.

## How it works

- The player talks to Dylan and Jasmine via text input.
- Each response is generated by a local LLM (Ollama).
- The LLM controls character dialogue, emotional state, and story progression.
- Drama beats drive the narrative arc across the evening.
